# 松田 啓 — 油彩画家 ポートフォリオサイト

大阪府在住の油彩画家・松田啓の公式ポートフォリオサイトです。

https://matsuda-art.com

## 概要

作品ギャラリー、展覧会履歴、受賞歴、お知らせなどを掲載しています。

## 使用技術

- **Next.js 14** — Reactベースのフレームワーク
- **TypeScript** — 型安全な開発
- **CSS** — Tailwindを使わずフルスクラッチで実装
- **MDX** — お知らせ記事の管理
- **Formspree** — お問い合わせフォームの送信処理
- **Vercel** — デプロイ・ホスティング

## 主な機能

- 作品ギャラリー（モーダル拡大表示・前後ナビゲーション）
- スマートフォン対応（レスポンシブデザイン）
- お知らせ記事（MDXファイルで管理）
- お問い合わせフォーム
